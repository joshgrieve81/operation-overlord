# Test1 — Orbiting Case Study Field

Lofi experimental portfolio prototype for a Creative / Product UX Designer.

## Concept

A spatial portfolio where placeholder case-study images orbit the viewer in Three.js. The visitor scrolls through editorial sections while the archive reacts dynamically: cards drift, tunnel rings pass by, camera depth changes, and the field feels like a discoverable product-design atmosphere rather than a static grid.

## Run it

```bash
cd "/root/Documents/Hermes projects/Test1"
npm install
npm run dev
```

Then open the Vite URL shown in the terminal.

## Build

```bash
npm run build
```
